STEPS to create vite project with reactStep1: npm create@latestStep2: follow the promptsStep3: cd <project-name> you just entered while following prompts.Step4: npm install to install required pluginsStep5: npm run dev or npx vite to run the project.Host same project on mobile Method1: use (npm run dev -- --host)Method2: add (“host”: “vite --host” inside scripts of package.json) <-- this might not work alwaysMethod3: add server:{    host: true,}, inside defineConfig function.
